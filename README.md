# College Placement Management System

A database management system developed to efficiently manage student placement activities, company records, and recruitment processes.

## Overview

This project focuses on designing and implementing a relational database-driven placement management system using SQL and MySQL. The system supports student, admin, and company-related operations for organized placement management.

---

## Features

- Student and company record management
- SQL-based CRUD operations
- Authentication and login functionality
- Relational database schema design
- Advanced SQL queries and joins
- Normalization and efficient data retrieval

---

## Technologies Used

- SQL
- MySQL
- PHP
- HTML/CSS
